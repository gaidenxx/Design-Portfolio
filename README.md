# Wireframes Figmajam
  A wireframe is a simple visual step outline of a webpage or app that I can use to shows the layout and structure before start the prototypes and to get all the main UX from users and technical points from team. It helps me plan where elements will go and how to make this feature work in a technical way.

# Prototypes Figma & AdobeXD
